# 共演・自主企画戦略

**作成日**: 2026年5月  
**目的**: ファン層の重なりと規模感から最適な共演相手を選定し、チケット動員と認知拡大を最大化する

---

## 共演相手の選定フレームワーク

ツーマン・スリーマンの共演相手は以下の2軸で評価する。

```
         ファン層の重なり
               ↑ 高い
               │
  死んだ僕の彼女 │ Luby Sparks  ← 理想ゾーン
  cephalo      │ Homecomings
               │
───────────────┼─────────────── 規模差
  小さい        │              大きい
               │ ラブリーサマーちゃん
               │
               ↓ 低い
```

**理想ゾーン**: ファン層の重なりが高く、規模差が適切（0.5〜5倍）な相手

## フェーズ別 共演ロードマップ

### Phase 1（2026年内）：実績を積む

#### ① Luby Sparks × MoritaSaki ツーマン【最優先推奨】

**なぜ今やるべきか**
- Spotify月間リスナーがほぼ同規模（27,400 vs 22,500）で動員リスクが低い
- どちらも日本語 vs 英語という差別化があり、セットリストの被りがない
- 両バンドともSpotify「Haze」（日本Shoegazeプレイリスト）圏内 → 同じリスナーが両方聴いている
- Luby SparksはHomecomings・DYGLとの共演実績あり → MoritaSakiへの箔付けになる

**想定規模・収支**
```
キャパ: 200〜250（東京または大阪）
チケット: ¥3,500
動員目標: 200人
粗売上: ¥700,000 + 物販¥80,000 = ¥780,000
経費: -¥350,000（会場・PA・ゲスト費含む）
想定利益: ¥430,000
```

---

#### ② cephalo × MoritaSaki ツーマン【同期枠】

**なぜやるべきか**
- 2023〜2024年デビューの「同世代Shoegaze」という物語性
- 「nevv you」などShoegazeZINEコミュニティで既に同一シーンに存在している
- cephaloはiTunes Alternativeチャート9位の実績があり、集客力がある
- どちらのファンにとっても「発見」になる

**想定規模**
```
キャパ: 150〜200
東京・大阪各1本
自主企画として打てる最初の2本柱
```

---

#### ③ 死んだ僕の彼女 × MoritaSaki ツーマン【コアファン・海外訴求枠】

**なぜやるべきか**
- MoritaSakiが影響元として公言している → 「師弟共演」という物語がある
- Bandcamp・海外ShoegazeコミュニティでのDead Boyfriend（死んだ僕の彼女）の評価は高く、海外在住ファンが来場する
- 規模は小さいが、コアなShoegazeファンへの訴求力は圧倒的
- 海外メディアへの英語プレスリリースの題材になる（「日本Shoegaze2世代の邂逅」）

**想定規模**
```
キャパ: 100〜150（小さめの箱でソールドを狙う）
東京1本
海外ファン・インディーメディア向け話題性が高い
```

---

### Phase 2（2027年）：ステップアップ

#### ④ Homecomings × MoritaSaki ツーマン【同郷京都枠】

**なぜやるべきか**
- 同郷・京都という強いストーリーラインがある
- Homecomings（75,900リスナー）はMoritaSakiの3.4倍 → 適度な上方への引き上げ効果
- Homecomingsのファン（Dream Pop/Indie層）はMoritaSakiのターゲットと完全に重なる
- felicityレーベル（国内Dream Pop界の重要レーベル）への接触機会にもなる

**想定規模**
```
キャパ: 300〜400（京都または東京）
ソールド目標でPhase 1からの成長を証明する
```

---

#### ⑤ ラブリーサマーちゃん × MoritaSaki ツーマン【大抜擢枠】

**なぜ2027年以降か**
- リスナー数252,600人（MoritaSakiの11倍）→ 今の規模では動員の非対称性が大きすぎる
- 2026年内に200枚完売の実績を積んでから交渉するほうが説得力がある
- For Tracy Hyde（Shoegaze）との文脈から、ラブリーサマーちゃんのファン層はMoritaSakiと親和性が高い
- **渋谷WWW（500cap）クラスの会場でソールドを狙える最初の相手**

**想定規模**
```
キャパ: 400〜500（渋谷WWWクラス）
チケット: ¥4,000
動員400人 × ¥4,000 = ¥1,600,000
物販込み粗売上: ¥1,760,000
これが実現すれば「ネクストステージ」の証明になる
```

## 自主企画イベント設計案

### 企画A：「新世代日本Shoegaze」3マン

**出演**: MoritaSaki in the pool × Luby Sparks × cephalo  
**コンセプト**: 2020年代に台頭した日本Shoegazeの新世代3組

- **なぜ刺さるか**: 各バンドのファンが「もう1バンド発見」できる設計
- **海外訴求**: 英語歌詞(Luby)・日本語(MoritaSaki・cephalo)の混在が外国人来場者に響く
- **想定会場**: 東京・大阪各1本、キャパ300〜350
- **主催**: MoritaSaki Entertainment（自主企画として箔がつく）

---

### 企画B：「日本Shoegaze縦断」3マン

**出演**: 死んだ僕の彼女 × MoritaSaki × cephalo  
**コンセプト**: 2000年代〜2020年代、日本Shoegazeの系譜

- **なぜ刺さるか**: 影響元・現役・新世代という縦断ラインに物語性がある
- **海外訴求**: Bandcamp・Reddit等の海外ShoegazeコミュニティへのPRに最適
- **想定会場**: 東京、キャパ150〜200（完売前提）

---

### 企画C：「京都インディー」企画

**出演**: MoritaSaki × Homecomings × 京都発バンド1組  
**コンセプト**: 京都発インディーの現在地

- **なぜやるか**: 地元京都での基盤確立。ローカルメディア・フリーペーパー掲載機会
- **想定会場**: 京都メトロまたは磔磔、キャパ250〜300
